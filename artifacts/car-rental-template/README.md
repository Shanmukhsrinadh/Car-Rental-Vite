# Car Rental Template

A premium car rental website template built with React + Vite. Dark theme, mobile responsive, WhatsApp booking integration.

## Tech Stack

- **React 19** + **Vite 6**
- **Tailwind CSS v4**
- **Radix UI** components
- **Wouter** routing
- **Framer Motion** animations
- **TypeScript**

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:5173](http://localhost:5173)

## Build for Production

```bash
npm run build
npm run preview
```

## Customization

Edit `src/data/content.ts` to update:
- Business contact info (phone, WhatsApp, email, address, Google Maps)
- Vehicle fleet (name, category, price, specs, features, images)
- Services offered
- Testimonials

## Sections

- **Hero** — full-screen landing with booking search form
- **Vehicles** — filterable fleet grid with WhatsApp booking
- **Services** — service offerings and trust features
- **Testimonials** — auto-scrolling customer reviews
- **Gallery** — photo grid
- **Contact** — embedded map with CTA buttons
